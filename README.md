# home-leisu-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。每个页面均为独立的静态文件，不依赖于特定域名或后端服务。

## 项目简介

本仓库主要用于存放各类静态 HTML 页面，适合作为学习笔记、工具页面、演示页面或临时存档的集合。页面之间彼此独立，无需额外配置即可在浏览器中直接打开查看。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（如 CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，命名清晰以便识别。
- `assets/`：公共资源文件，按需引用。
- 更多目录结构可根据实际内容调整。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接通过文件系统或静态服务器访问。
- 页面内容可能包含个人笔记、工具、演示或临时存档，不针对特定网站或服务。
- 页面之间无依赖关系，可单独复制使用。

## 维护说明

- 页面更新请直接修改对应文件，并保持目录结构清晰。
- 如需新增页面，请在 `pages/` 下创建新目录或文件，并确保引用资源路径正确。
- 本仓库不提供在线预览或部署配置，仅作为本地或自托管使用。
